PROJECT 03

프로젝트 3 : UI/UX DESIGN

플랫한 일러스트와 컬러감으로 단순하지만 눈에 잘 들어오는것을 목표로 디자인하였습니다.
일러스트를 적극활용하여 딱딱하고 어려운느낌보다는 어플이용이 쉽고 편안하게 보이는
부분을 강조하였습니다. 기존 서점어플과는 다르게 사용자의 정보와 책의 선호도를 조사하여
추천 서적을 보여주고, 다양한 장르, 에디터의 의견과 추천을 보여주는 '사용자 중심 서점'을 목표로 브랜딩하였습니다.

사용 TOOL

HTML / CSS / FIGMA / MEDIA QUERY

기존의 모바일 서점 어플리케이션은 책에 대한 방대한 데이터를 이용자들에게 편하게 보여주기 위해 디자인적인 부분보다는 가독성,
편의성을 기준으로 제작되어 있습니다. 그리고 구매하려는 도서를 간단하게 찾을 수 있도록 개발되어 있습니다. 하지만 디자인과 기존편리성을 접목시킬 수는 없을까?
단순 베스트셀러가 정렬된 메인페이지가 아닌 사용자가 원하는 맞춤형 UI/UX 로는 만들기 어려울까? 하는 생각에서부터 브랜딩 되었습니다.
square는 어플리케이션 이용자의 경험과 구매의도에 맞추어 구성하는데 초점을 맞추고 심미성을 강조한 사용자중심의 어플리케이션입니다.
