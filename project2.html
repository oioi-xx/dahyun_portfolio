프로젝트 2 : 렉서스

Lexus 의 기존 적응형 페이지를 반응형페이지로 디벨롭하였습니다. 기존 적응형 사이트를 clone coding 하여 태블릿, 모바일, PC 환경에도
깨짐이나 잘리는 부분없이 반응하도록 만들었습니다. 기존 고정 사이드바를 모바일버전에서는 하단으로 위치하게 만들었고,
이미지를 max-width 를 적용시켜 화면비율에따라 반응하도록 디자인하였습니다.

사용 TOOL

HTML / CSS / FIGMA / MEDIA QUERY
